rscl.connect {big.data.table}R Documentation

Connect to Rserve instances

Description

A wrapper to 'RS.connect' function using 'sapply' over recycled input arguments. It also requires data.table package.

Usage

rscl.connect(port = Sys.getenv("RSERVE_PORT", "6311"),
  host = Sys.getenv("RSERVE_HOST", "127.0.0.1"), tls = FALSE,
  proxy.target = NULL, proxy.wait = TRUE, pkgs = character())

Arguments

port

integer vector of port numbers

host

character scalar, soon will support vector.

tls

logical, see '?RSclient::RS.connect' for details.

proxy.target

logical, see '?RSclient::RS.connect' for details.

proxy.wait

logical, see '?RSclient::RS.connect' for details.

pkgs

character vector of packages to load after connect to node.

Value

List of active connections to Rserve nodes.
