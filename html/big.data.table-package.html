big.data.table-package

Description

Leverage data.table onto set of Rserve nodes.

Details

Use 'as.big.data.table' methods to extract/load/assign/populate data to a list of R node connections. Methods for 'function' and 'call' will be evaluated remotely, method for 'data.table' will be evaluated locally and assigned to nodes.
